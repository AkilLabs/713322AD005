### Installation

1. Clone the repository:
```
git clone <repository-url>
cd Round1
```

2. backend
Django(python)
```
cd backend
```
change the AUTH_TOKEN
```
python manage.py runserver
```

3. frontend
Next.js(tsx)
## Getting Started


```bash
cd frontend
npm i 
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
